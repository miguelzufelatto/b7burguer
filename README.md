# 🍔 B7Burguer

Uma landing page moderna e responsiva para uma hamburgueria virtual. Este projeto simula a interface de uma plataforma onde os clientes podem pesquisar, visualizar ofertas e fazer pedidos de hambúrgueres online.

🔗 **[Acesse o projeto online aqui](https://miguelzufelatto.github.io/b7burguer/)**

## 📝 Sobre o Projeto

O **B7Burguer** é uma página web estática (Front-end) desenvolvida para treinar e demonstrar habilidades na construção de interfaces modernas. O layout conta com uma seção Hero atrativa, uma barra de pesquisa e filtros para os produtos, além de uma vitrine em formato de grade (grid) exibindo os hambúrgueres disponíveis, seus preços e etiquetas de promoção.

## ✨ Funcionalidades

- **Design Responsivo:** A página se adapta perfeitamente a diferentes tamanhos de tela (Desktop, Tablet e Mobile), utilizando *Media Queries*.
- **Menu Mobile:** Layout adaptado com ícone de menu hambúrguer para telas menores.
- **Vitrine de Produtos:** Organização dos itens utilizando `CSS Grid`, garantindo um alinhamento perfeito.
- **Tipografia Customizada:** Uso das fontes *Calistoga* (para títulos e logo) e *Inter* (para textos), importadas do Google Fonts.
- **Semântica HTML:** Estruturação correta utilizando tags semânticas como `<header>`, `<nav>`, `<section>`, `<main>` e `<footer>`.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

- **HTML5**
- **CSS3** (Flexbox, CSS Grid, Variáveis e Media Queries)
- **Google Fonts**

O projeto segue uma estrutura simples e direta:

```text
├── assets/

```

## 🛠️ Como executar o projeto localmente

1. Faça um clone deste repositório:
   ```bash
   git clone https://github.com/miguelzufelatto/b7burguer.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd b7burguer
   ```
3. Abra o arquivo `index.html` no seu navegador de preferência ou utilize a extensão **Live Server** no VS Code.
